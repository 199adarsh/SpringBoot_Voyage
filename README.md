### ⟡ About

Welcome to **springboot-voyage**.

This repository is my learning journal for Spring Boot. Every folder, project, and commit represents a concept I've explored through building rather than just reading.

The goal isn't just to complete tutorials—it's to understand how production-ready backend applications are designed, implemented, and maintained.

---

### 🪐 Journey

- 🌱 Spring Boot Fundamentals
- 🌿 Dependency Injection & IoC
- 🍃 REST APIs
- 🌸 Spring Data JPA
- 🗄️ MySQL / PostgreSQL
- 🔐 Spring Security
- 📦 Validation
- ⚡ Exception Handling
- 🧩 DTOs & Mapping
- 📄 Pagination & Sorting
- ☁️ Docker
